Stock Tracker is a simple, interactive Python program designed to help users quickly calculate the total investment value of their selected stocks. The script comes with predefined prices for popular stocks including AAPL, TSLA, GOOG, and MSFT. It provides a user-friendly command-line interface where users can enter stock symbols and quantities, with built-in validation to ensure only valid inputs are accepted. Once the user finishes adding stocks by typing 'done', the program displays the total investment value in a clear and colorful output.
To get started, simply clone the repository and run the script using python stock_tracker.py. The program will first display the list of available stocks, then prompt the user to enter stock names and quantities one by one. It handles invalid stock symbols and non-numeric quantities gracefully, making it beginner-friendly and robust.
This project is ideal for anyone learning Python, particularly those interested in basic financial tools, user input handling, and simple calculations. In the future, the script can be enhanced by integrating live stock price APIs, adding portfolio saving features, or building a graphical user interface.
Feel free to customize the stock prices or expand the list as needed. Contributions and suggestions are always welcome!
